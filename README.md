# Islamabad House Price Prediction using Linear Regression (From Scratch)
Title:
House Price Prediction using Linear Regression (From Scratch)

Description:
This project implements Linear Regression from scratch using Gradient Descent to predict house prices in Islamabad. The model is trained on a custom dataset collected from Zameen.com using web scrapping.

Key Features:

Implemented Linear Regression without using ML libraries
Used Gradient Descent optimization
Mean Squared Error (MSE) as loss function
Evaluated model using RMSE and R-squared

Dataset:

Collected 50 house listings from Zameen.com
Features include:
Area (sq ft)
Number of rooms
Location
Other relevant attributes

Model Workflow:

Data collection and preprocessing
Feature selection
Train-test split
Model training using Gradient Descent
Evaluation on test data

Results:

Achieved reasonable prediction accuracy
Observed that area and location significantly impact price

Technologies Used:

Python
NumPy
Pandas
Matplotlib
